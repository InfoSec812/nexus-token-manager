Help

This is an application for managing token authentication to a Sonatype Nexus artifact repository. If you are able to authenticate to the Nexus server which this token manager is pointed at, you can manage your own tokens. If you have admin privileges on that Nexus server, you will also be able to manage other user's tokens from this application as well.
